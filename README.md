# natanparmigiano-portfolio

A simple personal portfolio website built with Vue, Vite, and Tailwind CSS.

**Live site:** [portfolio.natan.parmigiano.tech](https://portfolio.natan.parmigiano.tech)

Feel free to fork, copy, and modify this however you like — that's what it's here for. The content in `src/data/portfolio.js` is mine, but the structure and styling are yours to reuse.

## Development

```bash
pnpm install
pnpm dev
```

Build for production:

```bash
pnpm build
```

Deploy to Cloudflare (requires Wrangler setup):

```bash
pnpm deploy
```
